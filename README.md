# RAG for non-food bioeconomy policy analysis
This repository contains code supporting the manuscript on global non-food bioeconomy policy governance.
The code supports the following analyses:

1. Policy text preprocessing
2. Retrieval-based LLM classification
3. Sector, instrument, and SDG alignment analysis
4. Manual validation analysis

## Data availability

The original policy documents were obtained from publicly available policy repositories FAOLEX. Full original documents are not redistributed here due to third-party source document restrictions. Processed metadata, validation samples, classification outputs, and aggregated results will be provided where permitted.

## Notes

API credentials and third-party proprietary materials are not included.
